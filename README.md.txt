# Library Management System

## Description
This is a simple **Library Management System** built with **Java** and **MySQL**.  
It demonstrates basic **OOP concepts**, **JDBC database integration**, and **collections usage (ArrayList)**.  
Users can add books, register users, borrow and return books, and view available books and borrow history.

## Features
- Add books and users
- Borrow and return books
- View available books
- View user borrow history
- Data persistence using MySQL database

## Technologies Used
- Java (OOP, ArrayList)
- JDBC (MySQL database)
- MySQL (database)
